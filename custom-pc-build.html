Custom PC Build - My First Build

This project was my first experience building a custom PC from the ground up. I thoughtfully selected each component to create a well-balanced and efficient system that meets my needs. This build was a hands-on opportunity to deepen my understanding of hardware and the assembly process.

The build features components like an AMD Ryzen 5 5600X and an MSI GeForce RTX 3060, with a focus on both performance and aesthetics.
